Minecraft Bedwars Plugin

This is a plugin for the Java edition of Minecraft that recreates the popular game mode called "Bedwars". In Bedwars, multiple players or teams battle against each other, aiming to destroy each other's beds to prevent respawning after death.

Project Details

  • Developers: Adam and Fabian
  • GitHub Repository: https://github.com/AdamKITZ/SWP_Fabi-Adam

Setup Instructions

  1. Download the provided zip file.
  2. Extract the zip file, which contains another zipped file called "minecraft".
  3. Run the "00start.bat" file to start the server locally, or join the server at "kitzweb.com".
  4. In the game, click on the sign in front of you to teleport to the bedwars_lobby.
  5. Wait in the lobby for other players to join.
  6. When enough players have joined, a countdown will start, and you will be teleported to the actual game world (world_bedwars).
  7. Play the game, gather resources, and try to destroy other players' beds.

Plugin States

The plugin has different states that control the flow of the game:

  • Lobby State: Players join the lobby world and wait for more players to join.
  • Ingame State: The game starts, and players are teleported to their team spawns in the battle world.
  • Ending State: One team wins, the world resets, and the server is available for another round.

Involved Classes

The plugin contains several classes that handle different aspects of the game:

  • Commands: Contains logic for server administrator commands.
  • GameStateManager | LobbyState | IngameState | EndingState: Manage and handle game states.
  • Listeners: Handle events such as block breaks, player deaths, and world changes.
  • Object classes (e.g., Team): Store information about teams, players, and spawns.
