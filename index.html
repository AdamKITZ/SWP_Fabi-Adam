Project

Together, AdamKITZ and stampFabian are developing a plugin for the well-known Java game Minecraft: Java Edition. The plugin aims to recreate the popular game mode called "Bedwars" where players or teams battle against each other. The objective is to destroy other players' beds to prevent them from respawning after death.

First Setup

  1. Download the provided zip file.
  2. Extract the zip file, which contains another zipped file called "minecraft".
  3. You can either use this file and start "00start.bat" to join the server at "localhost".
  4. If you prefer not to set up the server on your PC, you can join the server hosted by Fabian at "kitzweb.com".

Gameplay

  1. After joining the server, you will be in the first world.
  2. Click on the sign in front of you to teleport to the "bedwars_lobby" where you wait for other players to join.
  3. A countdown will start, and once it's over, you will be teleported to the actual game world "world_bedwars".
  4. In the game world, you should be teleported to your team's color and receive a message stating your team.
  5. Gather resources and strategize to destroy other teams' beds and emerge victorious.
  6. Keep in mind that the plugin is still a work in progress, so more features will be added in the future.

Involved Classes

  • Commands: Contains logic for server administrator commands.
  • GameStateManager, LobbyState, IngameState, EndingState: Manages game states and their corresponding logic.
  • Listeners (e.g., blockBreakEvent, onPlayerDeath, playerWorldChangeEvent): Classes that handle events and perform actions based on server events.
  • Object classes (Team): Contains information about teams in the game, including color, players, and spawn points.
